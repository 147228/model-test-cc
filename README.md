## 项目简介

这是一个桌面GUI工具，帮助你快速测评任何AI模型的能力，并自动生成测评网站。无需修改代码，只需在界面上输入API配置即可开始测评。

### 核心特性

- 🖥️ **简洁的图形界面** - 无需命令行操作，所见即所得
- 🚀 **一键测评** - 自动执行所有测试用例，无需手动操作
- 📊 **实时进度显示** - 查看测试进度和详细日志
- 🎨 **自动生成网站** - 测评完成后自动生成精美的展示网站
- 🔧 **灵活配置** - 支持任意API端点和模型
- 📝 **内置测试用例** - 10个文生文 + 10个文生图测试案例
- ⚡ **多线程并发** - 支持最多30线程并发测试
- 💾 **结果保存** - 自动保存所有测试结果和生成的内容

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

或使用提供的启动脚本（会自动检查并安装依赖）：

```bash
# Windows
启动.bat
```

### 2. 启动程序

```bash
python main.py
```

或直接双击 `启动.bat`

### 3. 配置API

在界面上填写：
- **API URL**: 你的API端点（如 `https://yunwu.ai/v1`）
- **API Key**: 你的API密钥
- **文生文模型**: 文本生成模型名称（如 `gemini-3-pro-preview`）
- **文生图模型**: 图像生成模型名称（如 `gemini-3-pro-image-preview`）
- **并发线程**: 建议10-20

点击"保存配置"按钮保存设置。

### 4. 开始测评

1. 选择要测评的类型（文生文、文生图，或两者都选）
2. 点击"开始测评"按钮
3. 等待测评完成（进度条会显示实时进度）
4. 测评完成后会自动生成展示网站

### 5. 查看结果

- 点击"打开输出目录"查看所有生成的文件
- 展示网站位于 `output/website/index.html`
- 双击打开即可在浏览器中查看精美的测评结果

## 测试用例说明

### 文生文测试（10个案例）

测试AI模型的代码生成能力，包括：

- **视觉代码生成**: SVG动画、3D图形
- **交互游戏**: 俄罗斯方块、弹球游戏

- **视觉效果**: 粒子烟花
- **实用工具**: 计算器、时钟、Markdown编辑器

每个案例要求生成一个完整可运行的HTML文件。

### 文生图测试（10个案例）

测试AI模型的图像生成能力，包括：

- **未来科技**: 全息会议室、太空站
- **场景设计**: 赛博朋克街景、魔法森林
- **人物肖像**: 古风少女
- **奇幻生物**: 中国龙
- **建筑设计**: 现代建筑
- **产品设计**: 智能手表
- **美食摄影**: 日式拉面
- **自然场景**: 水下世界

每个案例使用专业的英文提示词确保最佳生成效果。

## 自定义测试用例

你可以修改 `test_cases/` 目录下的JSON文件来自定义测试用例。

### 文生文案例格式

```json
{
  "id": "T01",
  "name": "案例名称",
  "category": "分类",
  "difficulty": "难度",
  "prompt": "详细的提示词"
}
```

### 文生图案例格式

```json
{
  "id": "I01",
  "name": "案例名称",
  "category": "分类",
  "difficulty": "难度",
  "prompt": "Detailed English prompt for image generation..."
}
```

## 使用技巧

1. **首次测试**: 建议先用1-2个案例测试，确认API配置正确
2. **并发数设置**: 根据API限制调整，过高可能触发频率限制
3. **保存配置**: 配置好后记得保存，下次启动会自动加载
4. **查看日志**: 运行日志会显示详细的执行情况和可能的错误
5. **中断测试**: 可以随时点击"停止"按钮中断测试
6. **重新测评**: 再次运行会覆盖之前的结果

## 常见问题

### Q: 测试失败怎么办？

A: 检查日志输出，常见原因：
- API Key错误
- API URL不正确
- 模型名称错误
- 网络连接问题
- API配额不足

### Q: 可以测评其他模型吗？

A: 可以！只要API兼容OpenAI格式即可，包括：
- Claude
- GPT-4
- Gemini
- 国产大模型（通义千问、文心一言等）
- 任何兼容OpenAI API的服务

### Q: 测试用例可以修改吗？

A: 当然！直接编辑 `test_cases/` 目录下的JSON文件即可。

### Q: 生成的网站可以部署吗？

A: 可以！`output/website/` 目录下的文件是纯静态网站，可以直接部署到任何Web服务器或GitHub Pages。
