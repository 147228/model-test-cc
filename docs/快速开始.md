# 快速使用指南

## 第一次使用

### 1. 启动程序
双击 `启动.bat` 或运行 `python main.py`

### 2. 配置API（必须）
```
API URL:        https://yunwu.ai/v1
API Key:        你的密钥
文生文模型:      gemini-3-pro-preview
文生图模型:      gemini-3-pro-image-preview
并发线程:       10
```

点击"保存配置"

### 3. 开始测评
- 勾选"文生文测评"和"文生图测评"
- 点击"开始测评"
- 等待完成（10-20分钟）

### 4. 查看结果
- 点击"打开输出目录"
- 打开 `output/website/index.html`

## 目录结构

```
output/
├── text/           文生文结果（JSON + HTML）
├── image/          文生图结果（JSON + 图片）
└── website/        展示网站
    └── index.html  在浏览器中打开这个
```

## 测试用例

- **文生文**: 10个案例，测试代码生成能力
- **文生图**: 10个案例，测试图像生成能力

修改测试用例: 编辑 `test_cases/*.json`

## 常用操作

| 操作 | 说明 |
|------|------|
| 保存配置 | 下次启动自动加载 |
| 开始测评 | 执行所有测试 |
| 停止 | 中断正在运行的测试 |
| 生成网站 | 手动重新生成网站 |
| 打开输出目录 | 查看所有结果文件 |
| 清空日志 | 清空运行日志窗口 |

## 自定义测评其他模型

只需修改配置中的模型名称即可，例如：

**Claude:**
```
API URL: https://api.anthropic.com/v1
模型: claude-3-sonnet-20240229
```

**GPT-4:**
```
API URL: https://api.openai.com/v1
模型: gpt-4-turbo-preview
```

## 注意事项

1. API Key 必须填写正确
2. 首次测试建议并发数设为 5-10
3. 测评时间取决于模型响应速度和并发数
4. 结果会保存在 output 目录，再次运行会覆盖

## 快捷键

- **Ctrl+C**: 在命令行中中断程序（不推荐，建议用"停止"按钮）
