# 项目功能说明

## 项目概述

**AI模型一键测评工具** 是一个桌面GUI应用，用于快速测评AI模型的文生文和文生图能力，并自动生成精美的展示网站。

## 核心功能

### 1. 图形化界面操作

- 所有配置通过GUI完成，无需修改代码
- 实时显示测试进度和日志
- 一键启动测评流程

### 2. 灵活的API配置

支持配置任意兼容OpenAI格式的API：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| API URL | API端点地址 | https://yunwu.ai/v1 |
| API Key | 认证密钥 | sk-xxx... |
| 文生文模型 | 文本生成模型名 | gemini-3-pro-preview |
| 文生图模型 | 图像生成模型名 | gemini-3-pro-image-preview |
| 并发线程 | 同时运行的测试数 | 10-30 |

### 4. 多线程并发测试

- 支持1-30个并发线程
- 智能任务调度
- 自动重试失败的测试
- 实时进度更新

### 5. 结果自动保存

#### 文生文结果
- JSON格式的完整API响应
- 自动提取的HTML文件（可直接运行）

#### 文生图结果
- JSON格式的完整API响应
- 自动提取的图片文件（PNG/JPG）

### 6. 精美的展示网站

测评完成后自动生成响应式网站，包含：

**首页统计**
- 测试总数
- 分类统计
- 成功率显示

**文生文展示区**
- 卡片式布局
- 查看原始响应
- 直接预览HTML演示

**文生图展示区**
- 图片预览
- 提示词显示
- 响应内容查看

**交互功能**
- 点击放大图片
- 响应内容弹窗
- 键盘快捷键（ESC关闭）
- 移动端适配

### 7. 配置持久化

- 配置保存到本地JSON文件
- 下次启动自动加载
- 支持手动编辑配置文件

## 工作流程

```
1. 启动程序
   ↓
2. 配置API（首次使用）
   ↓
3. 选择测评类型
   ↓
4. 点击"开始测评"
   ↓
5. 多线程并发执行测试
   │
   ├─ 文生文测试 → 保存JSON → 提取HTML
   │
   └─ 文生图测试 → 保存JSON → 提取图片
   ↓
6. 自动生成展示网站
   ↓
7. 查看结果
```

## 技术架构

### 前端（GUI）
- **框架**: tkinter（Python标准库）
- **特性**:
  - 响应式布局
  - 实时日志显示
  - 进度条更新
  - 多线程安全

### 后端（测试引擎）
- **并发**: ThreadPoolExecutor
- **网络**: requests库
- **图像**: Pillow + base64解码
- **HTML提取**: 正则表达式

### 网站生成
- **纯静态**: HTML + CSS + JavaScript
- **无依赖**: 不需要服务器
- **响应式**: 支持各种屏幕尺寸

## 扩展性

### 自定义测试用例

编辑 `test_cases/text_cases.json`:
```json
{
  "id": "T11",
  "name": "你的案例名称",
  "category": "分类",
  "difficulty": "难度",
  "prompt": "详细的测试提示词..."
}
```

编辑 `test_cases/image_cases.json`:
```json
{
  "id": "I11",
  "name": "你的案例名称",
  "category": "分类",
  "difficulty": "难度",
  "prompt": "Detailed English prompt..."
}
```

### 修改网站样式

编辑 `website_generator.py` 中的CSS部分，可以自定义：
- 颜色主题
- 布局方式
- 动画效果
- 字体样式
